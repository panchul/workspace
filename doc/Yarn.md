## Installing 

---

```bash
brew install yarn
```

or

```bash
npm install -g yarn
```

Here's uninstalling:

brew:

```bash
brew uninstall yarn
```

tarball: 

```bash
rm -rf "$HOME/.yarn"
```

npm: 

```bash
npm uninstall -g yarn
```

ubuntu: 

```bash
sudo apt-get remove yarn && sudo apt-get purge yarn
```

centos: 

```bash
yum remove yarn
```

windows: 

```bash
choco uninstall yarn (or go to control panel > add/remove programs and uninstall it from there)
```
