
# screen
A terminal utility.

See Also:
 - [Linux](Linux.md)

---

Start screen:

    $ screen

To detauch press `Ctrl-A` and `d`.
To re-attach:

    $ screen -r

To see which sessions were saved:

    $ screen -ls
    1234 <timestamp> more info here
    
To pick one of them:

    $ screen -r 1234
