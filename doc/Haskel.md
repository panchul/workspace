## Installing

---

https://www.haskell.org/get-started/

To install `GHCup` on Linux, macOS, FreeBSD or WSL2,
run the following in a terminal (as a non-root user):

```bash
$ curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh
```
